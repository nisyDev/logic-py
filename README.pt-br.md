# Roadmap Python - Da Lógica à POO

[English Version](README.md)

Este repositório contém minha revisão de lógica aplicada ao Python, com exercícios práticos. Meu foco principal foi a transição da **Programação Estruturada** para a **Programação Orientada a Objetos (POO)**.

Desenvolvi um projeto em Inglês (ENG) e três projetos em Português (PT/BR).

---

## REPOSITÓRIO

O projeto está dividido em scripts funcionais (`.py`).

### 1. Lógica Aplicada
* **banco.py**: Um serviço de Banco 24h (PT/BR):
    * Validação de CPF (algoritmo real com dígitos verificadores).
    * Uso de `while True`, dicionários de menu e funções aninhadas.
    * Gestão de caixa, transferências e extrato, tudo simulado.
* **aplicação média de nota py.pdf**: Cálculo de média lógica aplicada a escolas com listas e loops (laços de repetição).

### 2. Programação Orientada a Objetos (POO)
* **poo_basic.py**: Fundamentos e conceitos de Objetos, Classes e Métodos de Instância (PT/BR).
* **OOP_intermed.py**: Conceitos avançados (ENG):
    * **Herança**: Classe `EnterpriseChannel` herdando de `Channel`.
    * **Composição**: Associações entre as classes `Video` e `Channel`.
    * **Decoradores**: Uso de `@property` para getters.
* **encapsulamento.py**: Simulação de proteção de dados e métodos de manipulação de dicionários como bancos de dados (PT/BR).
