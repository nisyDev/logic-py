# Roadmap Python - To Logic for POO

[README em Português](README.pt-br.md)

This repository contains my review in logic applied to python, with exercises. My main focus was on **Structured Programming** to **Object-Oriented Programming (OOP)**.

I have developed one project in English (ENG) and three projects in Portuguese (PT/BR).

---

## REPOSITORY

The project is separated in functional scripts (`.py`).

### 1. Logic Apllied
* **banco.py**: A 24/7 Bank Service (PT/BR):
    * CPF Validation (real algorythm with verifying digits.).
    * `while True` uses, menu dictionary's and nested functions.
    * Cash management, transferences and extract, all simulated.
* **applying medium note py.pdf**: Media calculum in logic to schools applied with lists and loops.

### 2. Object-Oriented Programming (OOP)
* **poo_basic.py**: Fundamentals of Objects, Classes and Instance Methods (PT/BR).
* **OOP_intermed.py**: Advanced concepts (ENG):
    * **Inheritance**: Class `EnterpriseChannel` inheriting `Channel`.
    * **Composition**: Assossiations between `Video` and `Channel`.
    * **Decorators**: Using `@property` to getters.
* **encapsulamento.py**: Data protection silumation and manipulation methods of dictionarys with databases. (PT/BR)
